Project authors
==========

The application Horosc for Excel is based on John D. North's historical HOROSC program described in his book: John D. North, _Horoscopes and History_, London: Warburg Institute, 1986, appendix 4, pp. 197-218, and its 2021 adaptation to Google Apps Script, [Horosc for Google Sheets](https://github.com/Turbehrt/horosc-GoogleSheets)

The code has been entirely rewritten in VBA for Microsoft Excel by **François J. Tur** in 2025-2026.

**Alexandre Tur** provided scientific advice on the historical astronomy concepts used, conducted tests on historical material, and produced documentation.

Auteurs du projet
==========

L'application Horosc pour Excel est basée sur le programme historique HOROSC de John D. North décrit dans son ouvrage : John D. North, _Horoscopes and History_, Londres : Warburg Institute, 1986, annexe 4, pp. 197-218, ainsi que sur son adaptation de 2021 pour Google Apps Script, [Horosc pour Google Sheets](https://github.com/Turbehrt/horosc-GoogleSheets).

Le code a été entièrement réécrit en VBA pour Microsoft Excel par **François J. Tur** en 2025-2026.

**Alexandre Tur** a fourni des conseils scientifiques sur les concepts d'astronomie historique utilisés, a effectué des tests à partir de sources historiques et a produit la documentation.
